
# Introduction

This is a simple Bootstrap website designed to showcase the power and simplicity of Bootstrap framework. This website has been developed to demonstrate how you can use Bootstrap to create a responsive, user-friendly, and modern website.

## Deployed on

<https://webdevschool.netlify.app/>

## Getting Started

1. Clone or download the repository.

2. Open the index.html file in your browser to view the website.

3. Explore the source code and see how the different components and features of Bootstrap have been used.

## Built With

* Bootstrap - The web framework used
* Font Awesome - Icons library
